Python CLI Projects

This repository contains two simple command-line interface (CLI) applications built in Python: a Basic Stock Portfolio Tracker and an Enhanced Rule-Based Chatbot.

Prerequisites

To run these scripts, you must have:

Python 3.x installed on your system.

1. Basic Chatbot (chatbot.py)

Goal

To build a simple, rule-based conversational chatbot using functions and pattern matching.

Key Features

The chatbot uses a dictionary of patterns to generate responses. Key concepts include dynamic responses (for date/time) and random selection (for jokes).

How to Run

Save the code as chatbot.py.

Open your terminal or command prompt.

Run the script using the Python interpreter:

python chatbot.py


Available Commands (Try typing these in the chat)

Greetings: hello, hi, hey

Status: how are you

Facts: what is python, capital of france

Dynamic: time, date (Provides current date/time)

Fun: joke (Tells a random joke)

Exit: bye, goodbye, or exit
